Python Tkinter Calculator (Add/Subtract)

Как запустить
- Требуется Python 3.9+ (Windows/macOS/Linux)
- Запуск: `python big_calc_tk.py`

Возможности
- Два поля ввода чисел (положительные/отрицательные)
- Выбор операции: сложение или вычитание (радиокнопки)
- Диапазон ввода и результата: от −1 000 000 000 000.000000 до +1 000 000 000 000.000000
- Десятичный разделитель: точка или запятая (оба поддерживаются)
- Точная арифметика с Decimal, фиксированные 6 знаков после запятой, без экспоненциальной нотации
- Сообщение о переполнении, если результат вне диапазона
- Ctrl+C / Ctrl+V работают даже при русской раскладке (а также Ctrl+Insert / Shift+Insert)
- Правый клик: контекстное меню Копировать/Вставить/Вырезать
- После вычисления ввод не очищается — можно править частично
